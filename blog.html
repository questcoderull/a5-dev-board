Question-3: What is event delegation in the DOM?

Event delegation is when we add one event listener to a parent element instead of many listeners to child elements. It use event bubbling, where events move from the target element to the parent. This way, we can handle events for multiple child elements with one listener, making the code simpler and maybe faster.
